# YOUC 支援的國家和地區
生效日期：2026-06-15  
網站：https://ai.youc.online

本文件說明 YOUC 的服務可用地區、限制地區和地區合規要求。本文件是《YOUC 服務條款》的組成部分。由於上游服務商、支付通道、雲端服務、制裁、出口管制和各地監管要求會變化，YOUC 可隨時更新、限制或停止任何國家或地區的服務。

## 1. 總原則

YOUC 僅在同時滿足以下全部條件的國家和地區提供服務：

1. 該國家或地區不屬於中國大陸。
2. 你本人、你的組織、你的帳單地址、付款來源、IP 位址、裝置環境、主要營業地、實際控制人、最終使用者和實際用途均不在中國大陸。
3. 你所呼叫的上游服務商在該國家或地區允許存取、開發、商業使用和 API 用戶端提供。
4. 該國家或地區不受適用制裁、出口管制、禁運、反洗錢、反恐怖融資或其他交易限制。
5. 你的使用不需要 YOUC 在當地取得我們尚未取得的許可、備案、註冊、安全評估、演算法備案、生成式人工智慧服務備案、支付牌照、行業資質或其他行政批准。
6. YOUC 未基於合規、風控、安全、支付、上游要求或商業判斷將該國家或地區列為限制地區。

如果上述任一條件不滿足，該國家或地區即不受支援，即使網站可以被存取、帳號可以註冊、訂單可以建立或支付可以完成，也不代表 YOUC 向該國家或地區提供服務。

## 2. 中國大陸不受支援

YOUC 不面向中國大陸公眾提供服務，也不允許從中國大陸存取、購買、加值、呼叫、轉售、整合、推廣或實質性使用服務。

你不得將 YOUC 用於在中國大陸提供或變相提供以下服務：

1. 生成式人工智慧服務。
2. 深度合成服務。
3. 演算法推薦服務。
4. 網際網路資訊服務。
5. API 轉接、模型呼叫轉售、額度分發、共享訂閱、付費呼叫、代充代付或類似服務。
6. 任何可能需要中國大陸行政許可、ICP 備案、經營性 ICP 許可、演算法備案、安全評估、生成式人工智慧服務備案、內容安全審核、個人資訊保護合規、資料出境合規、支付結算合規或稅務合規的服務。

如果你位於中國大陸，或你的產品主要面向中國大陸使用者，或你以中國大陸為主要經營地、註冊地、居住地、受益地、帳單地、付款地、伺服器地、行銷地或最終使用者所在地，你不得使用 YOUC。你使用 VPN、代理、境外伺服器、境外網域、境外手機號碼、境外支付帳戶或他人身分繞過本限制的，全部後果由你自行承擔。

## 3. 上游服務商地區要求

YOUC 的不同管道可能依賴不同上游服務商。你必須同時滿足對應上游服務商的支援國家和地區要求。上游政策嚴於本文件的，以更嚴格規則為準。

| 上游或功能 | 地區判斷規則 |
| --- | --- |
| OpenAI API 相關管道 | 僅限 OpenAI 官方 API 支援的國家和地區；在 OpenAI 未支援地區存取或提供存取可能導致帳號被封鎖或暫停。 |
| Anthropic / Claude 相關管道 | 僅限 Anthropic 官方支援國家和地區及其支援地區政策允許的場景。 |
| Google AI Studio / Gemini API 相關管道 | 僅限 Google 官方 Gemini API / AI Studio 可用地區；如在 EEA、瑞士或英國向使用者提供 API 用戶端，還需遵守 Google 對付費服務的特別要求。 |
| 其他模型、支付或雲端服務管道 | 以對應服務商最新條款、地區列表、制裁、出口管制、行業限制和帳戶規則為準。 |

YOUC 不保證某一地區對所有模型、所有上游、所有支付方式或所有功能均可用。某個模型、管道、功能、支付方式或帳號可用，不代表其他模型、管道、功能、支付方式或帳號也可用。

## 4. 預設不支援地區

除非 YOUC 以書面形式明確允許，以下地區、主體或場景不受支援：

1. 中國大陸。
2. 任何上游服務商未支援、暫停支援、禁止存取或禁止提供存取的國家和地區。
3. 適用制裁、禁運、出口管制或交易限制覆蓋的國家、地區、政府、實體、個人、船舶、機構或最終用途，包括但不限於相關制裁規則適用時的古巴、伊朗、北韓、敘利亞、克里米亞、頓內茨克、盧甘斯克、塞瓦斯托波爾及其他被列入限制名單的地區或主體。
4. 需要 YOUC 取得當地許可、備案、牌照、監管批准或本地化義務而 YOUC 尚未取得的地區。
5. 高詐欺、高拒付、高濫用、高監管風險或支付通道不穩定地區。
6. 你無法提供真實身分、真實所在地、真實付款來源、真實最終使用者或真實最終用途證明的任何情形。

我們可將任何地區、ASN、IP 區段、支付方式、電子郵件網域、手機號碼區段、裝置指紋、使用者群體或業務場景納入限制範圍。

## 5. 香港、澳門、臺灣及其他特別地區

香港特別行政區、澳門特別行政區、臺灣地區及其他具有特殊監管或上游可用性差異的地區，不因不屬於中國大陸而自動獲得全部服務支援。

你在上述地區使用服務時，仍必須滿足：

1. 所呼叫上游服務商明確支援該地區。
2. 支付方式、帳單地址、帳號註冊地和實際使用地符合上游及支付通道要求。
3. 服務不得主要面向中國大陸使用者，不得用於向中國大陸提供 API 轉接、付費呼叫、生成式人工智慧、深度合成或類似服務。
4. 不得以該地區主體、伺服器、網域、代理、支付管道或電話號碼規避中國大陸限制、上游地區限制或其他合規要求。

## 6. 地區識別與證明

為判斷地區合規，我們可綜合使用以下資訊：

1. 註冊資料、登入 IP、呼叫 IP、裝置資訊、User-Agent、時區、語言、ASN、代理/VPN/資料中心特徵。
2. 帳單地址、付款帳戶、支付通道、銀行卡 BIN、收款/付款地區、拒付記錄。
3. 手機號碼、電子郵件網域、組織資料、營業地、KYC/KYB 文件、稅務資訊。
4. 伺服器位置、應用程式名稱、網站內容、主要語言、行銷管道、客戶支援記錄、最終使用者分布。
5. 上游服務商、支付機構、雲端服務商、風控供應商、權利人、監管機關或執法機關提供的資訊。

如果我們無法確認你符合地區要求，或合理懷疑你規避地區限制，我們可拒絕服務、限制呼叫、要求補充證明、凍結餘額、暫停訂閱、關閉帳號或拒絕退款。

你應持續確保你的所在地、組織所在地、最終使用者所在地、業務投放地區、付款來源和實際用途符合本文件。若你的地區狀態發生變化，或你的產品開始觸及不受支援地區，你必須立即停止相關使用並通知我們。
