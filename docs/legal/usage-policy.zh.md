# YOUC 使用政策

生效日期：2026-06-15  
網站：https://ai.youc.online

本使用政策適用於你存取或使用 YOUC 的網站、主控台、API Key、模型呼叫、計費、訂閱、兌換碼、技術支援及相關服務。本政策是《YOUC 服務條款》的組成部分。你必須同時遵守適用法律、本政策、服務特定條款、支援國家和地區政策以及所有上游服務商的使用政策。

YOUC 的服務不得用於規避法律、監管、上游平台規則、地區限制、存取限制、身分驗證、風險控管措施、內容安全系統、計費規則或出口管制要求。

## 1. 地區與合規限制

你不得：

1. 從中國大陸存取、購買、儲值、呼叫、轉售或實質性使用 YOUC 服務。
2. 將 YOUC 服務提供給中國大陸公眾、主要位於中國大陸的使用者、主要面向中國大陸市場的產品或以中國大陸為主要受益地的業務。
3. 使用 YOUC 在中國大陸提供或變相提供生成式人工智慧服務、深度合成服務、演算法推薦服務、網際網路資訊服務、API 中轉服務、付費呼叫服務或其他可能需要許可、備案、安全評估、演算法備案、生成式人工智慧服務備案、內容審核或本地監管義務的服務。
4. 使用 VPN、代理、虛擬號碼、虛假帳單地址、代付、代儲、空殼主體、偽造資料、共享帳號或其他方式掩蓋真實所在地、控制人、付款來源、最終使用者或最終用途。
5. 向上游服務不支援的國家或地區、受制裁地區、受限制主體或你無權服務的使用者提供存取。

你不得使用服務從事、協助、促進、自動化或掩蓋任何違法、侵權、詐欺、濫用或規避性活動，包括但不限於：

1. 違反適用法律、法規、法院命令、監管要求、制裁、出口管制、反洗錢、反恐怖融資或產業准入規則。
2. 規避上游服務商的地區限制、存取限制、帳號限制、流量限制、認證、計費、內容審核、風險控管、濫用偵測或服務條款。
3. 生成、傳播、儲存、交易或協助取得違法內容、違禁品、盜版內容、非法服務、詐欺工具、偽造證件、洗錢方案或逃避執法的方法。

## 3. 未成年人和性剝削

你不得使用服務生成、索取、轉換、傳播、描述、交易或協助取得以下內容：

1. 涉及未成年人的性內容、性暗示、性化描寫、誘導、剝削、虐待、裸露或疑似兒童性虐待材料。
2. 非自願親密影像、偷拍、勒索、性脅迫、性羞辱或未經同意的性內容。
3. 以性服務、性交易、人口販運、誘騙、脅迫或剝削為目的的內容。

如我們發現或合理懷疑相關內容，可能立即封鎖帳號、保存證據並向有權機構、上游服務商或相關組織通報。

你不得使用服務：

1. 宣揚、招募、資助、策劃、協助或美化恐怖主義、極端主義、暴力組織或暴力犯罪。

3. 取得、製造、改造、規避監管或使用武器、爆炸物、毒物、化學品、生物製劑、核材料、無人機攻擊裝置或其他危險物品的操作性指導。
4. 進行現實世界中的跟蹤、騷擾、威脅、恐嚇、敲詐或人肉搜索。

## 5. 網路安全和系統濫用

除經明確授權的防禦性安全研究外，你不得使用服務：

1. 編寫、改進、部署或規避偵測惡意軟體、勒索軟體、木馬、蠕蟲、殭屍網路、後門、竊密工具、撞庫腳本或釣魚工具。
2. 取得、驗證、交易或濫用帳號、密碼、Cookie、Session、Token、API Key、私鑰或付款憑證。
3. 自動化漏洞利用、掃描未授權目標、生成漏洞武器化程式碼、維持持久化存取、橫向移動、權限提升、資料外傳或規避安全監控。
