01 · 语言切换（LocaleSwitcher）

现状：面板与触发器边框重叠（margin-top: -2.5px 或 inline 虽有 6px 但被 detached 模式抵消）； hover 为 var(--youc-soft) 浅灰。

现状
触发器底边 ↔ 面板顶边
重叠，无可见间隙
  • hover「English」：浅灰底 #ecebe4，不是黑底
  • 选中「中文」：白底黑字，与 hover 区分弱
  • 列表紧贴触发器，硬阴影被「吃掉」一部分
修改后（建议）
触发器底边 ↔ 面板顶边
6px 间隙，面板独立浮动
  • hover：深灰底（ink 68% 混 paper），比选中浅一档
  • 选中：纯黑底 var(--youc-ink)，对比更清晰
  • margin-top: 6px + 5px 硬阴影完整可见

鼠标移入右侧选项可体验 hover

02 · Select 分组选择（带搜索）

Portal 浮层定位目前用 top: rect.bottom - 2.5px，导致与触发器拼接； 全局 !important 规则也可能压过 hover 色。

现状
搜索…
  • hover 行：浅灰 --youc-soft
  • 选中行：仍是白底（当前代码），与 hover 几乎一样
  • 搜索框与选项区整体贴住触发器
修改后（建议）
搜索…
  • 未选中 hover → 深灰反色（略浅于选中）
  • 已选中项 → 纯黑底，与 hover 可区分
  • 面板整体下移 6px，与触发器视觉分离

鼠标移入选项可体验 hover

03 · 生产样式 token（已落地）

以下规则已写入 frontend/src/style.css 与 YoucDropdown.vue。

/* style.css — tokens */
--youc-dropdown-hover-bg: color-mix(in srgb, var(--youc-ink) 68%, var(--youc-paper));
--youc-dropdown-hover-fg: var(--youc-bg);
--youc-dropdown-selected-bg: var(--youc-ink);
--youc-dropdown-selected-fg: var(--youc-bg);

/* hover：深灰反色；selected：纯黑/纯白反色 */

/* YoucDropdown.vue — 浮层定位 */
style.top = `${rect.bottom + 6}px`
