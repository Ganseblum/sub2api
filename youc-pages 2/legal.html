服务条款

最后更新 · 2026-06-01

1 · 服务说明

YOUC 是一个 AI API 网关平台，将多家 AI 产品的订阅额度统一分发与管理。用户通过平台生成的 API Key 访问上游 AI 服务，平台负责鉴权、计费、负载均衡与请求转发。

2 · 账户与安全

您需对账户下生成的所有 API 密钥及其调用行为负责。请妥善保管密钥，切勿在客户端代码或公开仓库中明文暴露。如发现密钥泄露，请立即在控制台吊销并重新生成。

3 · 计费规则

平台按实际 Token 用量精准计费。账户余额不足时相关密钥将被暂停调用。具体单价以「渠道定价」页面公示为准，平台保留根据上游成本调整定价的权利。

4 · 使用限制

禁止利用本服务从事任何违反法律法规或上游服务条款的活动，包括但不限于滥用、批量爬取凭据、制造恶意软件等。违规账户将被限制或封禁。

5 · 免责声明

本服务以「现状」提供，上游服务的可用性、响应内容由第三方厂商决定，平台尽力保障稳定但不对由不可抗力或上游中断造成的损失承担责任。
